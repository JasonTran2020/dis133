Fabflix

Demo video of Fabflix being stress test in load-balancing configuration

A full-stack web project centered around a database containing thousands of actors, movies, directors, and their corresponding relationships. A logged-in user may browse/search movies based on different criterias such as movie title, actor name, year, genre, etc. The project is powered by technologies such as Java Servlets, MySql, and Apache Tomcat. SuperPrice

Gif of demoing some features of SuperPrice

My first independent project for an Android App. This app allows a user to search for particular grocery items, in which the app will show products matching the user's search from various supermarkets. Items selected will be placed into different carts, grouped by the particular supermarket location, allowing the user to see which markets are the closest and has the cheapest overall price. Obviously relies on the Android SDK, but also used Back4App to store user data online and the Kroger API to get product details at particular locations.
